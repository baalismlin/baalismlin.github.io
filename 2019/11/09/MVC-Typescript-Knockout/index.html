ASP.NET MVC + Typescript + Knockout

废话

前段时间要把一个很老的 win form（windows CE）程序重写成移动端，项目虽然难度不大，但逻辑步骤较为繁杂，如果用纯js写，不知道要走多少坑，好在微软又搞了个ts。至于knockout，公司一直在用，也够用，如果用angular，vue感觉有点牛刀小用。

搭建

项目例子可以在我的GitHub上找到,以下是具体的搭建流程。

首先用vs创建一个 asp.net mvc 的项目，

然后在Scripts目录下创建一个新目录TypeScript，我们的ts文件就写在这。

tsconfig里的 moduleResolution 要设置为 node，types 要添加 knockout， 如果要用 jquery 就添加 jquery。

在TypeScript目录下运行 npm i 安装依赖包
